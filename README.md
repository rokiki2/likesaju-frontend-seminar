# likesaju-solution

### 2학기 세미나 프론트엔드 스켈레톤 코드

이 레포지토리는 2학기 세미나를 위한 프론트엔드 스켈레톤 코드입니다. 이 프로젝트는 React를 기반으로 하며, Tailwind CSS를 사용하여 스타일링되었습니다.
